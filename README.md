# macOS-Terminal-Resource-Monitor
a macOS Terminal Resource monitor that gets and prints out systems resource information to the terminal 

Specters Terminal System Resource Monitor Displays...

- Displays CPU usage
- MEMORY usage
- TOP 5 Memory Intensive running Tasks
- System Uptime
- List of ALL Running Tasks by ID
